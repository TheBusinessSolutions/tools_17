## List View Order

Are users frequently requesting to change the order of columns in the Odoo list view?
Worry not! Each user can now set their own preferred order of columns in the list view.
They can even select which columns should be visible and which should appear under optional
columns. If no custom order is configured, the system will display the default column order. 
For models with multiple list views, the column order settings are saved separately for each view. 
Additionally, the user can reset the column order to the default settings at any time.

## Usage

1. Install Module.
2. Give Modify List View group to users who should be able to reorder their list views
3. Give Modify List View (All Fields) to users who should be able to see all fields to reorder
